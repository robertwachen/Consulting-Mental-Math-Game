# Consulting Mental Math Game
By Robert Wachen and Noah Schiff

Simple application created using Java (Swing API for UI) to help prepare students for mental math problems that could appear on consulting case interviews.

Types of Problems:
  - Percentage Problems (What is 15% of 30B?)
  - Division Problems (What is 300M divided by 20K?)
  - Multiplication Problems (What is 25K times 40K?)
